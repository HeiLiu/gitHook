## pre-commit
客户端`hooks`，在`git add`提交之后, 执行 git commit 键入提交信息前执行 `pre-commit`钩子，此时可以对将要提交的代码进行代码检查、优化代码格式（`lint` 、 `prettier`风格检查）。
可以用`git commit --no-verify`绕过该环节。
